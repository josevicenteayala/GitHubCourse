# Step 05 — Documentation Generation

## Objective

Use Copilot to produce clear function docstrings and usage documentation.

## Copilot Feature Focus

- Documentation generation

## Task

1. Add complete docstrings to functions in `exercise.py`.
2. Fill `USAGE.md` with one quickstart example and one edge-case example.

Docstrings must include:

- Purpose
- Parameters
- Return value
- One example

## Example Prompts to Try

- "Generate a concise Python docstring with args, returns, and examples."
- "Write a usage guide for this utility module for new developers."

## Expected Behavior

- Docs are technically correct and consistent with implementation.

## Validate

```bash
python3 scripts/evaluate.py --step step-05
```
